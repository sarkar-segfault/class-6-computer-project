HyperText Markup Language

HTML (HyperText Markup Language) is a markup language used to create websites. It is implemented in all browsers, including Google Chrome and Mozilla Firefox. An HTML document specifies the layout of all objects on a web page, using tags and attributes. HTML alone is insufficient for creating modern web apps, so it is complemented by CSS (Cascading Style Sheets) for styling the web page and JS (JavaScript) for manipulating the interface and various scripting workloads like server communication. HTML5 is the latest specification for the language, and will remain so for the foreseeable future, while CSS3 is the equivalent for CSS. JavaScript is specified by the ECMAScript standard.
